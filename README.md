## ğŸ Getting Started

Follow these steps to set up the project locally.

### 1. Prerequisites

Make sure you have Node.js and NPM installed.

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
# or if you are using Quasar CLI globally
quasar dev
```

The application will be available at `http://localhost:9000` (or the port specified in your console).

### 4. Build for Production

```bash
quasar build
```
